URL Shortener Microservice

Usage
URL Shortener Endpoint: https://api-projects-savageboy74.c9users.io/shorten/{url}

{url} must be a valid url to shorten.

Response: 
{
    "original": "{your_original_link}",
    "shortened": "https://api-projects-savageboy74.c9users.io/{unique_url}"
}
Shortened URL Endpoint: https://api-projects-savageboy74.c9users.io/{unique_url}

{unique_url} is the random characters generated for your original link, shown in the response above. This endpoint will redirect you to the original link.
